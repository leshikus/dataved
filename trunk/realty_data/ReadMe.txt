Установка модуля DEMO_8_2010_03_24_122208.zip

1. зайти в "Администрирование"
2. в нижней части странице в разделе "Студия" нажать на "Загрузчик модулей"
3. в строке "Модуль" указать путь к архиву модуля. можно использовать кнопку "Обзор..."
4. нажать кнопку "Загрузить", в таблице появиться запись с именем модуля и 2мя кнопками.
5. нажать "Установить"
6. Будет предложено ознакомиться с лицензией. отметить "Принимаю" и нажать кнопку "Вперед"
7. через некоторое время модуль установиться и будет выдано сообщение "Модуль установлен успешно"

----------

Импорт данных

1. установленный модуль скорее всего будет доступен в появившейся вкладке "Разное", в шапке страницы. нажать на "компания"
2. в боковом меню нажать "Импорт"
3. в "Укажите тип источника данных: " отметить "Текстовый файл с запятой в качестве разделителя" и в выпадающем списке выбрать "Двойная кавычка(")"
4. в "Параметры импорта" отметить "Создание записей".
5. нажать "Дальше >"
6. указать путь к файлу *.csv (см.Файлы демонстрационных данных для импорта), можно использовать кнопку "Обзор...". отметить галочку "Имеет заголовки: "
7. нажать "Дальше >"
8. нажать "Начать импорт"
9. в случае успеха будет сообщение "Готово: N записей успешно созданы".
10. в боковом меню нажать на "компании" для просмотра данных.

PS так же для всех остальных форм

----------

В данном модуле реализованы формы:   
 - организация
 - интерес
 - объект сделки
 - возможные сделки
 - компания
 - маркетинговый список
 - издания
 - предложение

----------

Файлы демонстрационных данных для импорта:

1. Компания			d1_Company.csv
2. Маркетинговый список		d1_The_marketing_list.csv
3. Издания			d1_Editions.csv
4. Интерес  			d1_interest.csv
5. Организация 			d1_com.csv
6. Объекты сделки  		d1_Object_of_the_transaction.csv
7. Возможные сделки		d1_The_possible_transaction.csv
8. предложение			d1_offer.csv

Рекомендуется импортировать в предложенном здесь порядке следования описания файлов.
